Inoffizielles Plugin für Tageslosung

== Description ==

Dieses Widget zeigt die Tageslosung der Herrnhuter Brüder-Unität. Die Daten liegen nicht im Plugin
sondern werden automatisch von [http://www.losungen.de/download/](http://www.losungen.de/download/)
herruntergeladen und eingebunden.

Das Widget zeigt ausschließlich deutsche Texte.
This Widget only shows text in german.

== Installation ==

1. Plugin zu Wordpress hinzufügen
1. Plugin Aktivieren
1. Plugin in Design -> Widgets zu einer Widget Area hinzufügen
